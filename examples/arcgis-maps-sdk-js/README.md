# Examples: ArcGIS Maps SDK for JavaScript

Context file examples for projects using the ArcGIS Maps SDK for JavaScript (`@arcgis/core`).

## Available examples

| Example | Stack | Description |
|---|---|---|
| [`vite-calcite/`](vite-calcite/) | Vite + `@arcgis/core` v5 + `@arcgis/map-components` + Calcite | Single-page mapping app with 2D/3D views, deployed to GitHub Pages |

## What belongs here

Any project using `@arcgis/core`, `@arcgis/map-components`, or the ArcGIS REST JS SDK — regardless of framework. Examples could include:

- `react-arcgis/` — React + `@arcgis/core`
- `scene-viewer/` — focused on 3D SceneView patterns
- `feature-editing/` — editing workflows with attribute rules and domains
- `offline-first/` — disconnected / air-gapped deployment patterns
