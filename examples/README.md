# Examples

Real-world AI context file examples, organized by technology.

Each example is a `CLAUDE.md.example` (or equivalent) from an actual project. Copy and adapt for your own.

---

## Categories

| Category | Description |
|---|---|
| [`arcgis-maps-sdk-js/`](arcgis-maps-sdk-js/) | ArcGIS Maps SDK for JavaScript projects |

Future categories could include `calcite-design-system/`, `native/` (iOS, Android, .NET, Qt), `experience-builder/`, etc. — just create the folder when there's a real example to add.

---

## Contributing an example

1. Pick the right category folder, or create a new one if none fit
2. Create a subfolder named after the project type (e.g. `react-arcgis`, `scene-viewer`)
3. Add a `CLAUDE.md.example` (or `AGENTS.md.example`, `copilot-instructions.md.example`)
4. Optionally add a short `README.md` explaining the project setup

Scrub any real project names, URLs, or internal details before committing.
